Bitwise Operators

OK, we are getting a little serious now. If you don't understand this don't worry too much. I have been programming since I was 6 years old and I have never had a use for bitwise operators.
That was a big statement to make, if bitwise operators are so useless why bother? The answer is becuase they are very, very fast compared to other methods. Doing Math on double types takes a lot of processing, if you are dealing with large scale operations like physics calculation your program may take over 10 times the time to run. Using bitwise operators on large int can seriously reduce the processing time. 

main() {
  int a;
  int b;
  
  a = 10010;
  b = 10001;
  print(a|b);

  a = 10010;
  b = 10100;
  print(a|b);

  a = 10010;
  b = 10010;
  print(a|b);
}

This output shows the results from doing a bitwise OR on the values. It's very easy to look at this an a number, for example 10,010 could be ten thousand and ten. however you should look at them as being a list of 0 and 1. Here you can follow some logic.
A: 10010
B: 10001
c: 10011

If you look at this in columns you can see that A1 has the value 1, A2 has the value 0. Therefore is A1 equal to 1, yes, therefore we place a 1
Is A2 equal to 1, No, is B2 equal to 1, no, therefore C2 equals 0.
Is A4 equal to 1, yes, therefore C4 equals 1.

Shift bitwise in Dart

Dart allows for bitwise operation (left/Right) on int variables. Remember our "OR" operator, this works by moving the value off to the left or right.
For example 10000000000 will become 01000000000 as we have moved the 1 to the right. This reduces the value from 1024 to 512. This is an amazingly quick way to divide by 2 as the CPU has very little to do. 

main() {
  int a;
  a = 1024;
  print(a);
  print(a>>1);
  print(a>>2);
  print(a>>3);
  print(a>>4);
  print(a>>5);
  print(a>>6);
  print(a>>7);
  print(a>>8);
  print(a>>9);
  
  a=1;
  print(a);
  print(a<<1);
  print(a<<2);
  print(a<<3);
  print(a<<4);
  print(a<<5);
  print(a<<6);
  print(a<<7);
  print(a<<8);
  print(a<<9);
}

Negative Numbers

Sometimes you want to convert a negative number to a positive value. This can happen when calculating a loan repayment. As the repayment is a negative number most consumers don't understand this concept, so the value is converted to a positive value. Google Dart has two methods for doing this, nagate (make something a negative number) and Abs for making something an absolute number. 

main() {
  int x = 5;
  print (x.negate());
  print (x.negate().abs());
}
